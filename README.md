--------------------
1. Опис сайту


"Фотогалерея" — це інтерактивна веб-платформа, розроблена для перегляду та обміну зображеннями, згрупованими за тематичними категоріями. Користувачі мають можливість зручно переглядати контент завдяки адаптивному дизайну, який підлаштовується під будь-який розмір екрану, від мобільних телефонів до десктопів. На сайті реалізовано функціонал завантаження власних фотографій через форму з клієнтською валідацією даних. Також доступна функція зміни візуальної теми (світла/темна), вибір якої зберігається між сесіями. Весь інтерфейс побудований на сучасних стандартах веб-розробки з акцентом на семантику та зручність користування.



--------------------
2. Структура проєкту



Проєкт має організовану файлову структуру, де стилі, скрипти та зображення винесені в окремі директорії:


Коренева папка /


- index.html — Головна сторінка з оглядом та останніми фото
- categories.html — Сторінка з переліком доступних категорій
- category_nature.html — Приклад сторінки окремої категорії
- upload.html — Сторінка з формою завантаження фотографій
- about.html — Інформаційна сторінка про проєкт
- contacts.html — Сторінка з контактною формою
- careers.html — Сторінка вакансій


/css


- style.css — Головний файл стилів, що об'єднує інші компоненти
- base.css — Базові налаштування, змінні кольорів, типографіка
- layout.css — Стилі для макету (Flexbox/Grid)
- components.css — Стилі для кнопок, форм та інших елементів


/js


- main.js — Основний файл скриптів (логіка інтерфейсу, валідація, теми)


/images


- Файли зображень (.jpg, .png) для контенту галереї



--------------------
3. Використані технології



HTML5


- Семантична розмітка (header, nav, main, footer, section)
- Форми та атрибути валідації (required, type, accept)


CSS3


- CSS Flexbox — Побудова навігаційного меню та вирівнювання елементів
- CSS Grid — Адаптивна сітка галереї зображень
- CSS змінні (Variables) для керування кольорами та темами
- Media Queries для адаптації під мобільні пристрої


JavaScript (ES6+)


- DOM Manipulation — Динамічна зміна контенту та стилів
- Event Handling — Обробка подій миші, клавіатури та форм
- LocalStorage — Збереження налаштувань теми користувача
- Клієнтська валідація даних форм



--------------------
4. Інструкція по встановленню

Для запуску проєкту на локальному комп'ютері необхідно клонувати цей репозиторій або завантажити архів з файлами. Розпакуйте папку та відкрийте її у вашому редакторі коду (рекомендовано VS Code). Запустіть файл index.html за допомогою будь-якого сучасного браузера або скористайтеся розширенням "Live Server" для коректної емуляції роботи сайту. Додаткового встановлення залежностей чи збірки проєкту не потрібно.
